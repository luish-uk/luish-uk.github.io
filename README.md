## What is this? 
This is my website linked to the domain csstudent.uk, where I hope to update (Semi-frequently) with information about new projects I am building as well as any other cool thing I get up to coding wise. I originally intend for this website to be extremely basic which is why it's full stack is in flask + sqlite. However i want to expand the stack with my experience to hopefully make it into a fully working website. 

## How will I accountably maintain the site?
I will log my progress on this website for others to see, the challenge is that I must do:
- 1 Hour of coding daily 
- Post my progress on this website 
I cannot skip a day, this project will encourage me to frequently change the website, for example when I first made it, it was very unsecure, and because it will be were I will post my progress daily, I must keep myself accountable for it's security.
